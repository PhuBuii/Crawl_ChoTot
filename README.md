# Crawl_ChoTot
This is repository that help you to collect information about car in ChoTot. You just clone this repository and download the model to predict color [here](https://drive.google.com/file/d/1sslZZErUNyMntfnWPiKlGe3xt8PMs-1d/view?usp=drive_link).After that run the jupyter notebook CrawData.ipynb to get car infomation.
